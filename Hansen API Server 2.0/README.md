Welcome to use Origin API Server.

This is a program that can help you build your own picture API
The specific usage is as follows:
1. Open the "photos" directory

2. Find the program "Hansen Photos Geter" and double-click to 
run it to get the pictures in the picture library. If you have
your own picture library, you can also change it to your own, 
but remember to keep those three very important files!. 
Remember to fill in the suffix of the picture correctly. If you
need to change the icon of the webpage, please change favicon.ico
to your own

3. Double-click to run the Hansen API Server program, and fill in 
the relevant data correctly

4. Start experiencing your own pictures API(/photos/index.html)!
